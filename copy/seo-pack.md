# SEO Pack

## Primary SEO Goal
Capture people searching for affordable professional portfolio website creation from LinkedIn or CV in Kenya.

## Target Audience
- Kenyan freelancers
- Job seekers
- Early and mid-career professionals
- Consultants and service providers

## Core Keyword Targets
- portfolio website from linkedin
- cv to portfolio website
- professional portfolio website kenya
- personal website for freelancers kenya
- build my portfolio from cv
- affordable portfolio website kenya
- linkedin profile to personal website
- freelancer portfolio setup kenya

## Secondary Keywords
- online cv website
- professional profile website
- portfolio link for job applications
- personal branding website kenya
- portfolio for remote work applications

## Homepage Metadata
**Title Tag:**
Professional Portfolio Website from LinkedIn or CV | WebPortfolio Kenya

**Meta Description:**
Turn your LinkedIn profile or CV into a professional portfolio website for Ksh 2,500. Get a portfolio URL you can share with clients and recruiters.

## Submission Page Metadata
**Title Tag:**
Submit LinkedIn or CV to Build Your Portfolio | WebPortfolio

**Meta Description:**
Submit your LinkedIn URL and/or upload your CV to start your portfolio build. Share your WhatsApp number and we will contact you with next steps.

## Thank You Page Metadata
**Title Tag:**
Thank You | Your Portfolio Request Has Been Received

**Meta Description:**
Your portfolio request has been received. We will contact you on WhatsApp and proceed with your Ksh 2,500 portfolio setup.

## Suggested URL Slugs
- `/` (Landing Page)
- `/submit` (Submission Form)
- `/thank-you` (Thank You)

## On-Page SEO Notes
- Include "Ksh 2,500" near the top of homepage content.
- Mention "LinkedIn" and "CV" in hero and process sections.
- Use one H1 per page.
- Keep testimonial names + roles present for trust signals.
- Add internal links:
  - Home CTA -> `/submit`
  - Submit success -> `/thank-you`
  - Thank You -> `/`

## Schema Suggestion
Use `Organization` schema on homepage and `Service` schema referencing portfolio setup service.

- "Professional portfolio website preview"
- "Freelancer sharing portfolio URL"
- "Submitting LinkedIn URL and CV form"
- "Thank you confirmation for portfolio request"
