# Design Direction

## Brand Feel
- Professional, sharp, and trustworthy
- Data-driven and business-focused
- Premium consulting tone without visual clutter

## Visual Language
- Clean high-contrast layout with strong hierarchy
- Modern card sections for services and process
- Balanced whitespace and concise content blocks

## Color Direction
- Primary: Deep navy for authority and credibility
- Accent: Electric cyan or teal for optimization/growth cues
- Neutral: White and cool gray backgrounds

## Typography Mood
- Bold, modern sans-serif for headings
- Highly readable sans-serif for body copy
- Strong spacing and readable line length for client scanning

## Media and Identity Rules
- Use provided client profile image in hero section.
- Use profile image-derived favicon.
- Use branded social card image for Open Graph and Twitter previews.
- If image is unavailable, activate no-image mode and keep layout balanced.

## UX Priorities
1. Quick credibility in first screen
2. Fast scan of services and process
3. Obvious contact actions (WhatsApp + LinkedIn)
4. Smooth mobile readability and tap targets

## Accessibility Notes
- Maintain semantic heading structure.
- Ensure adequate color contrast.
- Keep interactive elements keyboard accessible.
